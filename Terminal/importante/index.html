Comandos importantes para o terminal:

Olá caro visitante, bem, desde que mudei pro Debian, para aumentar conhecimento e compartilar isso a você, tive alguns conflitos, mas com o tempo eu aprendi e percebi que realmente se você quer uma distro linux difícil de se usar, ela não vai te ajudar igual o Mint, então aqui vai comandos do terminal bem importantes e que alguns você configura.

Aviso: Os comandos que estão aqui não servem no Arch Linux ou Fedora ou distros baseadas no Arch ou no Fedora, e sim apenas em Ubuntu/Debian ou distros baseadas no Ubuntu ou Debian.

flatpak:

O flatpak não vem junto com algumas distros linux, então ele deve ser instalado, e bem o que é o flatpak? O flatpak é um pacote de instalação de aplicativos, e sim ele é fácil de usar, tanto que eu sempre usei ele para facilitar nas instalações de aplicativos e agora aqui vai alguns comandos pra você saber:

Instalar: sudo apt install flatpak

Uma coisa que aprendi é que após instalar o flatpak, ele não vai funcionar vinculado a um repositório flathub, então use esse comando para resolver o problema:

flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

Comandos que você vai usar no flatpak normalmente:

flatpak install nome.do.aplicativo

flatpak uninstall nome.do.aplicativo

flatpak update

wget:

Pois é eu também fiquei impressionado quando soube que não vem junto a distro linux, mas é simples e funciona normalmente só usar esse comando:

sudo apt install wget

git:

Instalação: sudo apt install git

Eu sei que o git vem junto ao sistema operacional, mas muitas vezes acaba dando erro na instalação e não instala o git e bem é preciso instalar e o motivo é bem importante até, o git serve para você conectar seu sute no github pages ou até clonar um repositório github (Sim isso é para você desenvolvedor, mas use se quiser).

Login no git via terminal:

Login com email: git config --global user.email "seu@email.com"

Login com username: git config --global user.name "Seu Nome"

uv:

Bem o uv também é um pacote de instalação e que muitas ferramentas hackers usam como o theHarvester e aqui vai a instalação:

instalar: curl -LsSf https://astral.sh/uv/install.sh | sh

Reinicie o terminal: source ~/.bashrc

Agora para testar use: uv --version

Use isso se tiver erros: sudo apt install -y curl ca-certificates

go:

Instalação: sudo apt install golang-go

O golang-go pode ser muito utilizado em ferramentas hacking então se caso tu não tenha, recomendo instalar, já que ferramentas como WPProbe usam esse comando.

ruby:

Instalação: sudo apt install ruby-full

O ruby também é muito usado em ferramentas hacking, um exemplo disso é o XSpear que utiliza gem que é uma parte do ruby.

fasfetch ou neofetch:

Dois programas ótimos que mostram informações sobre seu pc para você, dizendo seu hardware e até sistema operacional e memória, bem, o neofetch foi descontinuado e então surgiu o fastfetch que dava informações a mais, então aqui vai:

Instalação:

Fastfetch: sudo apt install fastfetch

Neofetch: sudo apt install neofetch

Bem para usar os comandos é só os comandos neofetch (Se você instalou o neofetch) ou o fastfetch (Se você instalou o fastfetch)

Como colocar eles para aparecerem toda vez que você abrir o termina?

Primeiro você vai editar o arquivo do terminal: nano ~/.bashrc

Depois você vai na última linha de código que vai estar vazia e só colocar na última linha de código os comandos desses programas (No caso fastfetch ou neofetch)

Salvar e resetar o terminal: source ~/.bashrc
