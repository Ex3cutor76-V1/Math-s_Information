Ferramentas Web:

No caso por aqui só terá sites que podem ser usados como técnicas OSINT.

Osintframework: https://osintframework.com/

O Osintframework é uma ferramenta que lista outras ferramentas osint de forma organizada.

Osint.rocks: https://osint.rocks/

O osint.rocks também é uma ferramenta que possui outras ferramentas dentro, a única diferença dela e do osint framework é que o osint.rocks possui bem poucas ferramentas, mas ferramentas poderosas.

URL DNA: https://urldna.io/

Como citada no tópico #Como se proteger# ela pode ser usada para saber se um site é malicioso, mas sabe qual o interessante? Com essa ferramenta você também pode pegar informações sobre esse site como IP, domínio e outras informações.

Shodan: https://www.shodan.io/

O Shodan é uma ferramenta que pode te passar informações sobre um domínio coletando informações como IP, localização e até mesmo portas TCP/IP.

Tellows: https://www.tellows.com.br/

Esse site muita gente conhece já que com esse site é possível rastrear ou descobrir informações sobre um número de telefone.

Ipinfo: https://ipinfo.io/

Com essa ferramenta você pode ver informações sobre um IP.

Who.is: https://who.is/

Com essa ferramenta você pode ver informações sobre um domínio pegando informações como IP, DNS e etc.

Inflact: https://inflact.com/instagram-viewer/profile/

Com essa ferramenta você pode ver o perfil de usuário de alguém do instagram (Só pra avisar que ele não olha perfis privados, já que é de código aberto) e sim ele garante anonimato.

Webmii: https://webmii.com/

O webmii é uma ferramenta poderosa pra se caso você quiser buscar informações sobre uma pessoa como processos e até mesmo o que essa pessoa fez, só pesquisando o nome completo dela.

Mailscrap: https://mailscrap.com/check

Quer verificar se um email é real? Use essa ferramenta, ela te passará informações importantes.

Iplogger: https://iplogger.org/br/

Com esta ferramenta você pode criar um link falso de outros sites para assim caso a pessoa clique no link falso, irá mostrar informações como navegador, IP e até mesmo sistema operacional, além de outras informações interessantes.

Aware-online: https://www.aware-online.com/

Esse site na verdade é um curso Holandes de OSINT, mas nele possui uma variedade de ferramentas que podem ser utilizadas para investigação digital.

Data breach: https://databreach.com/

O data breach serve para você pesquisar se um email, número de telefone ou nome foram violados.

Have I Been Pwned?: https://haveibeenpwned.com/

A mesma coisa que o data breach, só que feito especificamente para emails.

Intelx.io: https://intelx.io/

Ele faz a mesma coisa que o data breach e o have i been pwned a única diferença é que ele tem mais outras formas de pesquisar alguma violação como domínio, email, IP, URL e etc.
