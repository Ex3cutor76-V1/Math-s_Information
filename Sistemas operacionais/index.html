Sistemas operacionais:

Antes de começar uma pequena explicação sobre o que é um sistema operacional:

Um sistema operacional é um software que permite a comunicação do usuário com os demais componentes do computador e suas funcionalidades, resumidamente é com o sistema operacional que você tem acesso a aplicativos e outros programas e softwares.
