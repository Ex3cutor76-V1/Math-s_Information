Linux:

Bem, só de você entrar aqui neste site já deve ter percebido que o Linux é o mais citado por aqui né? Pois é, o Linux é um dos sistemas operacionais daqui do site mais citados, tendo até mesmo tópicos só sobre o terminal, e bem o Linux na verdade não é só um sistema operacional ele é uma família de sistemas operacionais sendo muitas vezes chamadas de "Distros Linux" que bem sim o Linux também não é bem um sistema operacional, ele é um kernel de um sistema operacional e kernel é um núcleo de um sistema operacional que é a ponte principal entre o software (A parte lógica do computador, exemplo: Aplicativos) e hardware (A parte física do computador, exemplo: CPU, memória RAM, etc).

História:

Tudo se começou em 1991, quando um estudante finlandês chamado Linus Torvalds criou um novo kernel (núcleo) de sistema operacional por hobby, inspirado no Unix e no MINIX e o lançou como código aberto e bem esse projeto cresceu com a colaboração de uma comunidade global de desenvolvedores.

Curiosidades sobre Linux:

1) Sabia que o nome do pinguin que é o mascote do Linux, ele se chama Tux e bem o por que é um pinguin? Simples, porque Torvalds adorava pinguins além de que Torvalds já foi "mordido" por um pinguin no zoológico e o nome Tux combinad sendo T = Torvalds + U = Unix + X = Unix = Tux.

2) O Android utiliza o kernel Linux.

3) O Linux é código aberto ou seja qualquer um pode ver, modificar e distribuir o código, permitindo personalização e rápida evolução.

4) Mais de 90% dos supercomputadores mais poderosos do mundo rodam Linux, incluindo o acelerador de partículas LHC no CERN.

5) Além do Linux ser usado em supercomputadores ele é usado em servidores e IoT (Internet das coisas), sendo usado em TV's, servidores, roteadores e sistemas embarcados devido à sua estabilidade e adaptabilidade.
