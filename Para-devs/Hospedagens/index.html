Hospedagens:

Bem se caso você for fazer um site e quiser colocar ele na internet, você irá precisar de uma hospedagem, porque é com essa hospedagem que vai fazer o google reconhecer o seu site e deixar ele na internet.

Hostgator:

O Hostgator ele é bom pelo fato de oferecer planos com boa escabilidade e é um ótimo suporte com um uptime confiável, o único problema é que ele é pago.

Site oficial: https://www.hostgator.com.br/

Locaweb:

Uma coisa boa dessa hospedagem é o fato de possuir uma infraestrutura robusta e um soporte local português, mas pra nossa infelicidade ela é paga.

Site oficial: https://www.locaweb.com.br/

UOL host:

Se você é um pouco mais antigo na internet, você já conhece o UOL que para muitos é considerado o maior serviços de internet no Brasil por conta da sua facilidade no gerenciamento e com uma boa integração com outros serviços, mas assim como toda hospedagem, ele é pago.

Site oficial: https://uolhost.uol.com.br/

KingHost:

Reconhecido pela seu suporte dedicado e com um bom tempo de resposta e ainda por cima com servidores brasileiros, entretanto, ele é pago.

Site oficial: https://king.host/

DigitalOcean:

Conhecida pelo planejamento de alto desempenho, escapibilidade com preços bem acessíveis, ela é ideal para quem tem algum conhecimento técnico.

Site oficial: https://www.digitalocean.com/

Bluehost:

O lado bom do Bluehost é que ele é uma hospedagem rápida, possui uma boa integração com o wordpress e tem um ótimo suporte, entretanto, é pago.

Site oficial: https://www.bluehost.com/

Siteground::

Possui um suporte excelente, servidores rápidos e um bom desempenho geral, mas, também é pago.

Site oficial: https://world.siteground.com/

Wix:

Eu sei que você que nunca programou em toda sua vida como ser-humano já usou isso uma vez, e sim o lado bom do Wix é que ele é uma ferramenta fácil de se usar para criar e hospedar sites sem necessidade de habilidades técnicas e sim ele é pago.

Site oficial: https://pt.wix.com/

Hostinger:

Finalmente algo que realmente é bom para nós que somos de baixa renda não é mesmo? Sim o Hostinger é uma hospedagem boa por esses motivos aqui: Preço baixo, bom desempenho e suporte.

Site oficial: https://www.hostinger.com/

Outras formas de colocar seu site na internet de forma gratuita:

Github Pages:

Sim é possível colocar seu site na internet usando o Github, mas claro você terá que ter conhecimentos da ferramenta git para poder colocar seu site no Github Pages que inclusive é gratuito.

Curiosidade: O Math's information é um site que está dentro do github pages!

Netlify:

Uma plataforma de hospedagem que no caso vai permitir hospedar sites estáticos de forma gratuita, com funcionalidades avançadas por assim dizer como CI/CD (Integração contínua e entrega contínua), mas claro que a desvantagem é que para você usar as funcionalidades avançadas é preciso pagar um plano para liberar mais recursos de processamento.

Site oficial: https://www.netlify.com/

Google sites:

Com esse aqui além de criar o site do jeito que você quiser ele ainda te permite colocar seu domínio do jeito que quer e ainda por cima é muito fácil de usar, perfeito pra quem nunca programou na vida.

Site oficial: https://sites.google.com/u/0/?authuser=0

infinityFree:

Uma plataforma de hospedagem gratuita que permite você hospedar sites dinâmicos com PHP ou MySQL e sem anúncios, uma coisa boa também é que esse site possui recursos básicos o que acaba sendo bom para iniciantes.

Site oficial: https://www.infinityfree.com/
