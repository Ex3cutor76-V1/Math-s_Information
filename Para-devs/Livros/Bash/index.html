Bash:

Bash é um interpretador de comandos, usado mo Linux, Unix e até MacOS, ele basicamente interpreta comandos, e bem normalmente ele é usado para automatizar tarefas, manipular texto e até programação básica, mas também ele é usado para administrar sistemas, o bash é usado em servidores UNIX, automação de tarefas de DevOps, Deploy e manutenção de aplicação web, administração de sistemas e infraestrutura, scripts de build e integração interna contínua.
