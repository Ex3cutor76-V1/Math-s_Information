MatLab:

MatLab é basicamente uma linguagem de programação que na maioria das vezes serve para fazer cálculos matemáticos complexos, analisar dados, criar gráficos e simular sistemas, ele normalmente é usado em enegnharia elétrica, controle automação, processamento de sinais, inteligência artificial, robótica e economia quantitativa.
