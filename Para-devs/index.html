Certificado SSL (HTTPS):

O que é HTTPS?

A explicação também estará no tópico #Informações importantes#

Recomendações de certificado SSL/Como garantir um certificado SSL:

1)Escolha o tipo de certificado SSL sendo eles: Certificado SSL de domínio único para um domínio único (www.site.com), Certificado SSL Wildcard para proteger um domínio e todos os seus subdomínios (*.seusite.com), Certificado SSL Multi-Domínio para proteger múltiplos domínios com um único certificado (site1.com, site2.com, site3.com).

2)Escolha uma autoridade certificadora como por exemplo: Let's encrypt (gratuito), Comodo, DigiCert, GlobalSign.

links oficiais:

Let's encrypt: https://letsencrypt.org/pt-br/

Comodo: https://comodosslstore.com/

DigiCert: https://www.digicert.com/pt

GlobalSign: https://www.globalsign.com/pt-br

3)Solicite o certificado SSL (Sim a maneira como você solicita o SSL depende de onde o seu site está hospedado).

Maneiras de solicitar um certificado SSL:

a)Gerar uma solicitação de assinatura de certificado (CSR)

b)Encomendar o certificado

4) Instalar o certificado SSL no servidor

Formas de instalação: Algumas diretrizes gerais

a)Hospedagem compartilhada (Exemplo cPanel)

b)Hospedagem VPS ou servidor dedicado

5)Verificar instalação (Não é obrigatório, mas é sempre bom)

Ferramenta online para verificação de instalação de certificado SSL: https://www.ssllabs.com/ssltest/

6)Forçar o uso de HTTPS

Bem por aqui você vai ter que configurar um redirecionamento 301 em um arquivo .htacces (Se caso usar apache).

Exemplo de redirecionamento no .htacess:

RewriteEngine On

RewriteCond %{HTTPS} off

RewriteRule ^ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]

Agora se você utiliza Nginx:

server {

listen 80;

server_name www.seusite.com;

return 301 https://$server_name$request_uri;

}

Autentificação de 2 fatores: É um método de segurança que exige dois tipos de identificação para possuir acesso ao sistema.

Como fazer a autentificação de 2 fatores?:

Ative o 2FA no painel de administração de plataformas como Wordpress, Shopify ou Magento, ou também você pode utilizar plugins de segurança para implementar a funcionalidade.

Links de ajuda:

Wordpress: https://wordpress.com/pt-br/support/seguranca/autenticacao-em-duas-etapas/

Shopify: https://help.shopify.com/en/manual/your-account/logging-in/two-step-authentication

Magento: https://www.magenteiro.com/blog/magento-2/como-resetar-2fa-admin/

Backup Regular e Offsite: São cópias dos arquivos e banco de dados armazenadas em servidores ou serviços de armazenamento externo (off-site)

Como fazer isso?:

Configure backups automáticos diários ou semanais usando ferramentas como o UpdraftPlus ou também serviços de armazenamento na nuvem como por exemplo no Google Drive ou também no Proton Drive (Se caso você for um paranoico igual eu).

Bem essas são algumas formas bem eficazes de proteger seu site de ataques cibernéticos, espero ter ajudado!
