Maths-Password: Para Windows

Como que o Math's-Passwords funciona:

No caso ele gera uma senha aleatória com números, letras maiúsculas e minúsculas e caracteres especiais sendo eles: #*&

Instale as dependências no powershell (Mais recomendável):
winget install Git.Git
winget install Python.Python.3

Clone o repositório usando: git clone https://github.com/Ex3cutor76-V1/Math-s-Password.git

Entre no repositório: cd Math-s-Password

Executar: python .\Maths-Password.py ou python Maths-Password.py
