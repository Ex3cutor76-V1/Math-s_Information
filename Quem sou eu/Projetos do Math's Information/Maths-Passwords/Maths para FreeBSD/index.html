Maths-Password: Para FreeBSD:

Como que o Math's-Passwords funciona:

No caso ele gera uma senha aleatória com números, letras maiúsculas e minúsculas e caracteres especiais sendo eles: #*&

Instale as dependências:
sudo pkg update
sudo pkg install git
sudo pkg install python3

Clone o repositório usando esse comando: git clone https://github.com/Ex3cutor76-V1/Math-s-Password.git

Entre no repositório: cd Math-s-Password

Agora para executar use: python3 Maths-Password.py
