Math's-Passwords: Para Linux

Como que o Math's-Passwords funciona:

No caso ele gera uma senha aleatória com números, letras maiúsculas e minúsculas e caracteres especiais sendo eles: #*&

Como instalar e usar o Math's-Passwords:

Instalar as dependências acordo com sua distro:
Debian/Ubuntu ou distros baseadas: sudo apt install git python3
Arch ou distros baseadas: sudo pacman -S git python
Fedora ou distros baseadas: sudo dnf install git python3

Primeiro passo: Clone o repositório: git clone https://github.com/Ex3cutor76-V1/Math-s-Password.git

Segundo passo: Entre no repositório: cd Math-s-Password/

Agora para usar é só usar o comando: python3 Maths-Password.py e irá aparecer algo assim:

E pronto você tem agora uma senha forte.
