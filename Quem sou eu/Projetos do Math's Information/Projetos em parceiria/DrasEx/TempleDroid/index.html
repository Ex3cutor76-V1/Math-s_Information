Tutorial de instalação:

Vamos precisar dos requisitos primeiro:

Instalar o Termux

Instalar o wget: pkg install wget

Atualizar: pkg update && pkg upgrade -y

Instalar o root-repo (Caso não tenha): pkg install root-repo

Instalar o netcat: pkg install netcat-openbsd

Instalar o x11: pkg install x11-repo

Instalar o qemu: pkg install qemu-system-x86_64
