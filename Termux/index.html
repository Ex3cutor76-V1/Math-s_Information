Termux:

Já imaginou usar o Terminal só que no celular? Pois é, isso existe e tem como fazer usando esse programa chamado Termux.

O que é o Termux?:

Ele é basicamente um Terminal Linux só que para Android e IO's, ele faz as mesmas coisas que o Terminal faz, no caso mexer no sistema através de comandos, o lado bom de usar termux é que ele é poderoso para muitos fatores como hacking ético, controle absoluto do sistema e até mesmo mexer nos arquivos do sistema de forma profunda algo que softwares com GUI (Interface gráfica, não fazem normalmente)

Como instalar o Termux?:

Você pode instalar ele tanto pela play store quanto pela app store e até mesmo no F-Droid (Uma App Store só que open source)

Quando instalar use esses comandos para dar acesso a algumas coisas mais privilegiadas:

pkg install root-repo --> Com esse comando você consegue acesso a ferramentas que necessitam de root (Ou no caso administrador)

pkg install x11-repo --> Com esse comando você consegue acesso a repositórios com app's gráficos baseados em x11

termux-setup-storage --> O Termux pede permissão pro Android e libera o acesso do Termux ao armazenamento do celular, assim te permitindo mexer no sistema inteiro via comandos.

Ah e lembrando que é preciso dar pkg update && pkg upgrade para atualizar tudo, é sempre bom e funciona melhor quando se atualiza tudo.

Curiosidades interessantes:

O Termux usa o mesmo tipo de gerenciador de pacotes que o FreeBSD que é o pkg e sim aprendendo a usar o pkg no termux ajuda um pouco a entender o FreeBSD que sim também é um dos sistemas operacionais mais difíceis de usar, junto ao Linux e o Unix, mas claro que o Termux não ensina tudo ele só ajuda um pouco, mas não de fato significa que ele seja igual a ponto de ensinar, mas no mínimo ajuda.
