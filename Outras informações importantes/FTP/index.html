O que é um FTP? (File Transfer Protocol):

O FTP é como um tipo de ponte entre dois computadores, permitindo enviar e receber arquivos, com versões seguras para proteger os dados durante a transferência (Ele é como se fosse o carteiro).

Para que serve?:

Para receber e enviar arquivos de um computador para o outro, mas pode servir também para hospedar sites e backup remoto.

Tipos de FTP:

FTP Padrão (Não possui criptografia) --> Envia arquivos e senhas em texto simples.

FTPS (FTP Seguro) --> Um FTP com SSL/TLS, criptografando a conexão.
