O que é uma SSL? (Secure Sockets Layer):

É como se fosse um cofre digital que protege os dados enviados entre você e um site, no caso ele que garante o HTTPS.

Para que serve?:

Ele normalmente serve para manter os dados confidencial, integrado e auteticado.

Como conter um certificado SSL:

Bem... Normalmente ele deve conter: O nome do domínio, empresa responsável e chave pública para criptografar.
