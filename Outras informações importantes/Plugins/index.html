O que são plugins?:

O plugin é um complemento adicional que adiciona novas funcionalidades a um software, assim permitindo personalização ou expansão de recursos sem alterar o programa principal.

Para que serve um plugin?:

Para adicionar novas funções, melhorar segurança e até integrar serviços.

Exemplos de plugin:

Adblock (Do firefox)

Jetpack (Do Wordpress)
