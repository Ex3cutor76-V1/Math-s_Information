O que é um Proxy?:

É um intermediário ou mensageiro digital que envia pedido para a internet e devolve a resposta, podendo proteger, filtrar e até acelerar o tráfego.

Para que serve?:

Ele normalmente serve anonimato e privacidade, cache e velocidade, controle de acesso e segurança.

Tipos de proxy:

Proxy HTTP: Para sites e navegadores, no caso só pra web.

Proxy HTTPS / SSL: Suporta tráfego seguro e criptografado.

Proxy transparente: Usuário não percebe que usa proxy.

Proxy anônimo: Esconde o IP real (Tipo a rede tor)

Proxy reverso: Fica do lado do servidor, protege e distribui o tráego para múltiplos servidores.
