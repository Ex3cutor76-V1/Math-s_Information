O que é uma ASN? (ASN = Autonomous System Number):

Uma ASN é tipo uma identidade de rede na internet, a ASN permite que redes independentes saibam como se comunicar entre si, usando o protocolo BGP.

Tipos de ASN:

ASN pública: Usado para redes que se conectam a outras redes na internet o que permite que outras redes saibam alcançar seus IP's.

ASN privada: Usado internamente, não roteável na internet pública, usado para redes internas ou de testes.

Para que pode ser usado uma ASN?:

Ela pode ser usado para identificar rede na internet, roteamento de tráfego BGP, conectar redes entre si, controlar políticas de roteamento.

Como é um corpo de uma ASN:

ASN antigo: Valores de 1 a 65535 (Exemplo: AS3356)

ASN moderno: Valores de 0 a 4.294.967.295 (Exemplo: AS4200000001)
