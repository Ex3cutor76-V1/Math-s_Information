O que é uma CDN? (Content Delivery Network):

A CDN é uma rede de servidores espalhados pelo mundo, que trabalham juntos para entregar conteúdo de sites (Tipo imagens e vídeos, scripts ou até páginas), mais rapidamente e de forma confiável para os users.

Para que serve uma CDN:

Para reduzir a distância entre o user e o conteúdo do site.

Exemplos de CDN:

Cloudflare

Akamai

Amazon CloudFront
