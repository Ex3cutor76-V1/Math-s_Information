O que é um DNS? (Domain Name System):

O DNS é o sistema que irá traduzir o nome dos sites em IP's, permitindo que os computadores encontrem uns aos outros na internet de forma rápida e simples.

Para que serve?:

Ela serve para facilitar a navegação, rotear tráfego e gerenciamento de domínios.

Tipos:

Registros:

A --> Domínio para endereço IPv4 (Exemplo: 192.0.2.1)

AAAA --> Domínio para endereço IPv6 (Exemplo: O mesmo do A)

CNAME --> Cria um alias de outro domínio (Exemplo: www --> exemplo.com)

MX --> Define servidor de email do domínio (Exemplo: mail.exemplo.com)

NS --> Indica servidores autoritativos para o domínio (Exemplo: @ IN NS ns1.provedor.com)

TXT --> Armazena informações textuais (Exemplo: v=spf1 include:provedor.com~all")

SRV --> Define o serviço específico (Exemplo: O mesmo do TXT)

TTL --> Quanto tempo a informação pode ser guardada em um cache (Exemplo: 1 hora)
