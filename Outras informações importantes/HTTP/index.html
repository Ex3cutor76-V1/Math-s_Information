O que é um HTTP? (Hypertext Transfer Protocol):

É o protocolo que permite os navegadores pedir páginas e arquivos aos servidores e recebam respostas.

Para que serve?:

Com o HTTP você: pedi o conteúdo, recebe o conteúdo e interaja com o site.

Corpo de um HTTP:

http://domínio.com (Exemplo: http://google.com)
