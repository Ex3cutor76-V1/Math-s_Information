O que é um Mac Adress? (Media Acess Control Adress):

É basicamente o CPF da placa de rede do dispositivo, ele identifica de forma única na rede (No caso ele é único para cada dispositivo).

Para que serve?:

O Mac Adress serve para identificar dispositivos na mesma rede local, controle de acesso e rastreamento de hardware.

Corpo de um Mac Adress:

Código hexadecimal de 12 dígitos, normalmente ele é tipo assim, Exemplo: 00:1A:2B:3C:4D:5E
