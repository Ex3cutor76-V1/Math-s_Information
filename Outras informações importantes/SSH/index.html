O que é um SSH? (Secure Shell):

É como se fosse um controle remoto seguro para outro computador, tipo você controla um computador de forma segura o SSH, transferindo os dados de forma criptografada.

Para que serve?:

Normalmente o SSH é usado para acesso remoto seguro, execução de comandos, transferência segura de arquivos usando a base do SFTP e tunelamento (Com o SSH Tunnel).

Autênticação:

Senha --> Mais simples, mas menos segura.

Chave SSH (Isso é muito recomendado):

Chave pública --> Essa é a chave SSH que fica no servidor

Chave privada --> Essa é a chave SSH que fica no seu computador.
