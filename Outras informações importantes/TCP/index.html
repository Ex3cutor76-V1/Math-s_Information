O que é um TCP? (Transmission Control Protocol):

Ele é responsável por garantir que os dados sejam enviados corretamente pela rede, no caso ele trabalha junto com o IP para que todos os pacotes chegam corretamente sem erros.

Para que serve:

Ele garante que os dados chegam completos, sejam reenviados se algo falhar, cheguem em ordem correta e qu a conexão seja confiável.
