Como entrar na Darknet?

Bem quando nos referimos a deep web e dark web podemos utilizar o termo "Darknet" ou "Sites onion"

E bem para entrar na darknet é bem simples até, siga os passo abaixo:

Primeiro passo: Instale o tor browser

O tor browser é um navegador que basicamente tem contato com a rede tor e bem ele é perfeito para o anonimato, já que ele além de não salvar nada no histórico, você também fica quase irrastreável e bem o lado bom é que é com ele que vamos usar para entrar na darknet.

Aviso: se você não sabe o que é rede tor visite o tópico: #Rede tor#

O site oficial deles é: https://www.torproject.org/

Segundo passo: Tenha uma VPN

Bem uma VPN é essencial, mas pera o que é uma VPN? Aqui vai uma aula bem simples do que de fato é uma VPN: a VPN (Virtual Private Network) é um serviço que cria uma conexão segura entre um dispositivo e um servidor remoto criptografando os dados e mascarando endereços IP do usuário, então basicamente uma VPN serve para meio que "esconder" o seu IP colocando outro IP em cima.

Minhas recomendações de VPN:

ProtonVPN: https://protonvpn.com/

Terceiro passo: Tenha um antivírus

Eu sei que Antivírus quase não pode ajudar, mas vai por mim ele será bem importante no final, ainda mais que na deep web e dark web é um lugar bem sujo por assim dizer já que lá você pode facilmente ser hackeado ou pegar vírus.

Minhas recomendações de antivírus:

Kaspersky: https://www.kaspersky.com.br/

Clamav: https://www.clamav.net/ (Se caso você estiver no Linux assim como eu)

Malwarebyte: https://www.malwarebytes.com/pt-br/

Quarto passo: Pegue uma engine

Uma engine é basicamente um site onde irá conter links onions (No caso links da darknet) e bem eles são fáceis de encontrar na web normal que utilizamos um exemplo deles é o ahmia e hiddenwiki, abaixo estará o link deles pra você fazer buscas.

Ahmia: https://ahmia.fi/

Hiddenwiki: https://hiddenwiki.com/
