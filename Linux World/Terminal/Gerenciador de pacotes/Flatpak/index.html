Flatpak:

O flatpak é um sistema de pacotes universal no Linux, muito importante e até fácil de instalar aplicativos um exemplo disso é que a maioria dos usuários de Linux atualmente usam flatpak para facilitar a instalação de tal pacote (Um exemplo é que antes no Debian dava para instalar o steam usando o comando: sudo apt install steam, mas agora não funciona mais e graças ao flatpak ficou fácil para eu instalar o steam), o flatpak pode não vir em algumas distros mas ensinei lá no sub-tópico: Importantes a como instalar o flatpak e fazer ele funcionar.

E sim o flatpak ele instala pacotes normalmente isolado do sistema aumentando a segurança de não quebrar o sistema.
