DNF
Instalação: sudo dnf install <pacote>
Desinstalação: sudo dnf remove <pacote>
Atualizar pacote específico: sudo dnf update <pacote>
Atualizar todos os pacotes do sistema: sudo dnf upgrade
Procurar pacotes: sudo dnf search <termo>
Lista de pacotes: sudo dnf list installed
Informações do pacote: sudo dnf info <pacote>
