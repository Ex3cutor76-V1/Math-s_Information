APT
Instalação: sudo apt install nome-do-pacote
Lista de atualização: sudo apt update
Atualizar pacotes: sudo apt upgrade
Atualizar o sistema inteiro: sudo apt full-upgrade
Desinstalar: sudo apt remove nome-do-pacote
Desinstalar completamente: sudo apt purge nome-do-pacote
Remover arquivos restantes: sudo apt autoremove
Limpar arquivos de pacotes antigos: sudo apt autoclean
