APT:

O pacote de instalação "apt", muito usado no Debian ou Ubuntu e na maioria das distros linux que são derivadas do Debian/Ubuntu e APT significa Advanced Package Tool (Pacote Avançado de Ferramentas)

Comandos do Apt:

Dica de ouro: Se tvocê quiser ver a lista e atualizar tudo de uma vez use esse comando

sudo apt update && apt full-upgrade
