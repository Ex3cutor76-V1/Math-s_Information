Redes:

Olá caro visitante! Aqui e irei te ensinar alguns comandos para se conectar na internet wifi via terminal, então... Anote se quiser, mas preste bem atenção aos detalhes, é bem fácil.

nmcli dev wifi:

Bem, como podem perceber, usei o comando nmcli dev wifi para listar as redes wifi que vai mostrar coisas como nome de rede, BSSID, mode, chan, Rate e signal, mas também pode mostrar a segurança (Sendo WPA1 ou WPA2 ou outras) e algumas barrinhas mostrando o quanto a rede wifi é forte.

nmcli dev wifi connect "nome-da-rede" password "senha":

"Certo, e agora?" você pergunta depois de listar a rede wifi e relaxa! É bem simples!, o comando nmcli dev wifi "nome-da-rede" password "senha" vai servir para você se conectar a rede wifi, que está na lista claro e bem sim você vai ter que saber a senha do wifi, então... Melhor pedir pro seu vizinho antes de tentar hackear a rede wifi dele.

Explicação das coisas que aparecem no comando nmcli dev wifi:

In-use: Significa qual rede você está usando, então exemplo: Eu estou conectado no REPETIDOR, logo o meu in-use mostra uma estrela na linha do REPETIDOR.

BSSID: É o endereço MAC da rede wifi o que significa que é um identificador único.

SSID: Nome de rede

Mode: Tipo de rede.

Chan: É o canal utilizado.

Signal: Intensidade do sinal.

Bars: É a representação gráfica da intensidade do sinal.

Security: É o tipo de segurança (Sendo WPA1, WPA2 e etc.)

ifconfig:

O comando ifconfig serve para mostrar as configurações de rede, inclusive é com esse comando que mostra seu IP.

Aviso: Esse eu não irei mostrar imagem porque pode conter informações sensíveis como o IP.

iwconfig:

Com esse comando você pode configurar e visualizar parâmetros de interfaces de rede wifi.
